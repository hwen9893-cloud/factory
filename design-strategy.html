策略主线

把 README 与 2x、3x、5x、7x 文档压缩成一个可讲、可审、可落地的路线图。

阶段门

P0
市场验证有声书 / 动态漫优先；不写平台代码，只看真实反馈。
P1
最小自动化只自动化跑通的形式，优先知识库、小说链路和评测脚本。
P2
规模化生产扩展资产、一致性、音频或视频能力，并卡住 M3.5。
P3
IP 工厂化多形式、多平台、运营中心后置，避免早期过度抽象。

评测闭环

市场信号

完播率、追更、互动、涨粉、评论情绪决定是否进入自动化。

内容质量

设定冲突、人工修改率、角色量化分布约束 factory.novel。

一致性

身份、跨模态、时序、设定四个维度决定视频路线能否放行。

经济性

单章成本、GPU 成本、人工工时、收入趋势决定是否规模化。
