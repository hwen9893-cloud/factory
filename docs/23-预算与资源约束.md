# 7. 单集成本核算方法论（Unit Economics）

> "单集成本未核算清楚前不承诺更新频率"（§5）—— 本节定义如何核算。

## 7.1 成本结构（按形式）

### 形式 A：有声书/广播剧（章节）

```
单章成本 = LLM写作 + LLM审校 + TTS配音 + 人工工时

LLM写作   = 输入token数 × 输入单价 + 输出token数 × 输出单价
LLM审校   = 同上（审校 profile 通常更短）
TTS       = 字符数 × TTS单价（本地为0，云API按量）
人工工时   = (剧情修改 + 美工校对 + 代码维护) × 月薪 ÷ 产出章节数
```

### 形式 B：动态漫/短视频（分钟）

```
单分钟成本 = 小说成本（按字数折算）+ 图像生成 + LoRA训练摊销 + 视频生成 + 配音 + 人工

图像生成   = 图片数量 × GPU时长 × 云GPU单价
LoRA训练   = 训练时长 × 云GPU单价 ÷ 复用次数（摊销）
视频生成   = 视频秒数 × GPU时长系数 × 云GPU单价
```

## 7.3 可持续产出判断线

| 指标 | 判断方式 |
|------|---------|
| 变现 > 成本 | 当平台分成/创作激励 ≥ 月 Opex，即盈亏平衡 |
| 规模化临界点 | 单集成本随产量下降速度（学习曲线），判断何时值得投 Capex |
| ROI 正转 | 变现收入 / (Capex 摊销 + Opex) ≥ 1，启动规模化 |
